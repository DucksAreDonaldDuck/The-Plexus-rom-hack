# The Plexus

Hi everybody. This is the rom hack i'm making, called The Plexus. **Every copy of Super Mario 64 is personalized.** I just wanna clarify that this romhack is heavily inspired by the B3313 rom hack by *ChrisRLillo*. Huge shoutout to Chris and his team for making such a scary rom hack. Anyway, to compile just run `make and run.bat` for Windows users. Sorry but Mac isn't supported yet.
#
Also, **only the US rom is currently supported. However if you are a translator you can help us by making a pull request! :)**
